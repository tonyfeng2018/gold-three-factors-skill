# 黄金三因子打分系统

## 核心评分表

| 因子 | 满分 | 评分维度 |
|------|------|----------|
| 美元/实际利率 | /40 | DXY走势、Fed政策、实际利率变化 |
| 避险情绪 | /15 | 地缘冲突级别、停火协议、风险资产表现 |
| 滞胀逻辑 | /35 | CPI/PCE、油价、央行购金、实际利率前景 |

**总分 ≥72** → 买入信号  
**总分 ≥80 连续3天** → 加仓信号  
**4月22日停火谈判** → 无论总分多少，强制重打分并单独备注

---

## 数据获取

### 必读数据源（按优先级）

1. **Jin10 快讯数据库**（已有本地 SQLite）：
   ```bash
   node ./scripts/fetch_gold_data.py --today --important --json
   ```

2. **EIA 短期能源展望**（通过 Jin10 快讯获取）：重点关注 WTI/布油价格预期

3. **美元指数 DXY**：通过 `web_fetch` 抓取 Yahoo Finance 或通过经纪商数据

4. **黄金ETF持仓（SPDR）**：Jin10 快讯中有当日数据

5. **CPI/PCE 数据**：通过 Jin10 搜索

6. **央行购金动态**：Jin10 快讯搜索"央行+购金"或"黄金+增持"

### 快捷查询命令

```bash
# 今日重要快讯（Jin10）
node ./scripts/fetch_gold_data.py --today --important --json

# 过去24小时黄金相关快讯
node ./scripts/fetch_gold_data.py --hours 24 --json 2>/dev/null | python3 -c "
import json,sys
for item in json.loads(sys.stdin.read()):
    for kw in ['黄金','gold','XAU','美元','美联储','CPI','油价','EIA','央行','地缘']:
        if kw in item.get('content',''):
            print(f\"[{item['time']}] {item['content'][:300]}\")
            break
"
```

## 触发规则

### 定时任务
- **早报**：每日北京时间 08:00 前完成推送
- **晚报**：每日北京时间 22:00 前完成推送
- 使用 Jin10 数据库 + 实时网页数据综合打分

### 事件触发（立即重新打分）
- 停火谈判进展（美伊/以巴/中东任意）
- Fed 官员讲话（FOMC票委及以上）
- EIA 周度原油库存报告（每周三22:30）
- 央行购金数据发布（中国/俄罗斯/印度/波兰等）
- 地缘突发（战争扩大/外交破裂/制裁升级）

### 事件触发推送规则
若重新打分后总分 ≥72，立刻推送：
> 🚨 黄金短期上涨条件已满足，总分 **XX分**，建议立即买入
