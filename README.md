# Gold Three-Factors Scoring System
> 黄金走势三因子打分系统

一个基于**美元/实际利率**、**避险情绪**、**滞胀逻辑**三维度的黄金走势量化打分系统。适合机构交易员、宏观分析师和黄金投资者使用。

---

## 核心框架

| 因子 | 权重 | 数据来源 |
|------|------|----------|
| 美元/实际利率 | /40 | DXY、TIPS、Fed政策信号、沃什听证 |
| 避险情绪 | /15 | 地缘级别、VIX熔断阈值、48h动量衰减 |
| 滞胀逻辑 | /35 | CPI、WTI油价、央行购金、亚洲现货溢价 |

**总分 72+** → 买入信号  
**总分 80+ 连续3天** → 加仓信号

## 安装

```bash
# 克隆
git clone https://github.com/YOUR_USERNAME/gold-three-factors.git
cd gold-three-factors

# 设置 Jin10 数据库路径（可选，如使用本地数据）
# 编辑 fetch_gold_data.py，修改 JIN10_DB 为你的实际路径
```

---

## 快速使用

```bash
# 标准打分（示例参数）
python3 scripts/score.py 99.0 93.79 0.35 3.2 5 4 18 25 0 0 0 5

# 参数顺序：
# DXY WTI TIPS CPI 地带级别 央行购金 VIX 中国溢价 印度贴水 沃什 地缘h Fed信号
```

## 参数说明

### score.py 参数

| 参数 | 说明 | 示例 |
|------|------|------|
| DXY | 美元指数 | 99.0 |
| WTI | WTI原油价格（美元/桶）| 93.79 |
| TIPS | 10年期TIPS收益率（%）| 0.35 |
| CPI | 最新CPI年率（%）| 3.2 |
| 地带级别 | 地缘冲突级别 0-15 | 5 |
| 央行购金 | 央行购金信号 0-5 | 4 |
| VIX | VIX恐慌指数（≥35触发熔断）| 18 |
| 中国溢价 | 中国现货溢价（美元/盎司）| 25 |
| 印度贴水 | 印度现货贴水（美元/盎司）| 0 |
| 沃什 | 沃什听证会信号（-8~+8）| 0 |
| 地缘h | 地缘降级距今小时数 | 0 |
| Fed | Fed政策信号（-10~+10）| 5 |

### VIX 熔断阈值

| VIX 区间 | 得分 |
|----------|------|
| < 20 | 0 |
| 20-30 | +3 |
| 30-35 | +2 |
| **≥ 35** | **-10（熔断）** |

### 地缘动量扣分（48h修正）

| 降级情景 | 扣分 |
|----------|------|
| 战争→停火（24h内）| -10 |
| 战争→停火（48h内）| -7 |
| 冲突→停火（48h内）| -5 |

## 数据来源

- **美元指数 / TIPS**：Yahoo Finance / FRED
- **油价**：EIA / 金十数据
- **地缘动态**：金十快讯、Reuters、AXIOS
- **VIX**：CBOE
- **亚洲现货溢价**：Gold.org、Kitco、金十数据
- **央行购金**：IMF / 各央行官方公告
- **CPI/PCE**：BLS / Fred Economic Data
