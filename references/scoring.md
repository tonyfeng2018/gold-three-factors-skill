## 一、美元/实际利率因子（满分40分）

### 美元指数（DXY）评分

| DXY 区间 | 得分 | 说明 |
|----------|------|------|
| DXY > 108 | 0-5 | 美元强势，黄金承压最重 |
| DXY 103-108 | 5-15 | 美元偏强 |
| DXY 99-103 | 15-30 | 美元中性偏弱，黄金受益 |
| DXY < 99 | 30-40 | 美元弱势，创近期新低，黄金大幅受益 |

### 实际利率（10年期TIPS收益率）评分

| 实际利率 | 得分调整 | 说明 |
|----------|----------|------|
| 实际利率 > 2% | -5到-10 | 持有黄金机会成本高 |
| 实际利率 1%-2% | 0 | 中性 |
| 实际利率 0-1% | +5 | 黄金机会成本降低 |
| 实际利率 < 0% | +8到+15 | 负实际利率，黄金极受益 |

### Fed 政策信号评分

| 信号 | 得分调整 |
|------|----------|
| 降息预期升温（降息概率 > 60%）| +5到+10 |
| 按兵不动，声明偏鸽 | +3到+5 |
| 按兵不动，声明偏鹰 | -3到-5 |
| 加息预期升温 | -5到-10 |

### 美联储换届风险（补入，4月16日沃什听证会前后）

| 情景 | 得分调整 |
|------|----------|
| 沃什听证会偏鸽派（支持当前政策）| +5到+8 |
| 沃什听证会鹰派（主张紧缩+缩表）| -5到-8 |
| 听证会前不确定性期 | +3到+5 |
| 听证会通过/结果不确定 | 根据市场定价调整 |

> **凯文·沃什（Kevin Warsh）背景**：被视为通胀鹰派，主张缩减资产负债表。4月16日当周参议院确认听证会是关键节点。

### 综合评分参考

**强买入区间（35-40分）**：DXY < 99 + 实际利率 < 0% + 降息预期升温  
**中性区间（15-25分）**：DXY 100-105 + 实际利率 0.5-1.5%  
**偏空区间（0-10分）**：DXY > 106 + 实际利率 > 2% + 加息预期

## 二、避险情绪因子（满分15分）

> ⚠️ 本因子含两项修正：**VIX熔断阈值**（新）+ **地缘动量扣分**（新）

### 地缘冲突级别（基础分）

| 状态 | 基础分 | 说明 |
|------|--------|------|
| 中东/欧亚大规模战争，全面升级 | 12-15 | 最高级别避险 |
| 区域冲突持续（不停火） | 8-11 | 强避险 |
| 停火谈判进行中，结果不确定 | 4-7 | 中等避险 |
| 停火协议达成但脆弱 | 2-4 | 轻量避险 |
| 完全和平，无重大地缘风险 | 0-2 | 无避险溢价 |

### 地缘动量衰减扣分（48小时修正）⚠️ 新增

**触发条件**：地缘状态在48小时内从"军事行动"降级为"停火/谈判"

| 降级幅度 | 额外扣分 | 说明 |
|----------|----------|------|
| 从"大规模战争"→"停火谈判"（24-48小时内）| -8到-10 | 多头踩踏风险极高 |
| 从"区域冲突"→"停火谈判" | -5到-7 | 避险溢价快速消退 |
| 从"谈判"→"和平协议" | -3到-5 | 尾部风险下降 |

**判断方法**：对比今日与昨日 Jin10 快讯中地缘相关描述，识别降级信号

### VIX 熔断阈值（替代原VIX加分规则）⚠️ 升级

| VIX 区间 | 得分 | 说明 |
|----------|------|------|
| VIX < 20 | 0 | 市场平静 |
| VIX 20-30 | +2到+4 | 适度恐慌，避险资金流入黄金 |
| VIX 30-35 | +1到+2（上限）| 恐慌加剧但尚可控 |
| **VIX ≥ 35** | **-8到-12** | 🚨 **熔断**：流动性危机，机构被margin call，无差别抛售黄金换取现金 |

> ⚠️ **触发熔断时的预警格式**：
> `🚨 VIX熔断触发(XX)！警惕无差别抛售，若金价同步下跌则进入防御模式`

### 市场情绪辅助指标

| 指标 | 看避险 | 看风险偏好 |
|------|--------|------------|
| VIX 20-30 | ✅ | |

## 三、滞胀逻辑因子（满分35分）

### 通胀数据（CPI/PCE）

| CPI 年率 | 得分 | 说明 |
|----------|------|------|
| CPI > 5% | 25-35 | 严重滞胀，黄金最强逻辑 |
| CPI 3%-5% | 15-25 | 温和滞胀 |
| CPI 2%-3% | 8-15 | 接近目标但仍支撑 |
| CPI < 2% | 0-8 | 通胀温和，黄金支撑弱 |

### 油价（WTI 原油现货）

| WTI 价格 | 得分调整 | 说明 |
|----------|----------|------|
| WTI > $120 | +8到+10 | 严重通胀压力 |
| WTI $100-$120 | +4到+8 | 显著通胀 |
| WTI $80-$100 | 0到+4 | 中性 |
| WTI $60-$80 | 0 | 通胀温和 |
| WTI < $60 | -3到-5 | 通缩风险（反而利空黄金） |

**注意**：布伦特现货溢价（Backwardation）是更超前的信号。
- 现货溢价 > $20/桶 = 即期供应极度紧张（历史极端）
- 现货溢价 > $30/桶 = 前所未有，暗示短期价格将剧烈波动

### 实际利率前景（与滞胀联动）

| 情景 | 得分 |
|------|------|
| 滞胀（通胀高 + 增长弱 + 不加息）| 30-35 |
| 通胀高企 + 经济强 + 不加息 | 25-30 |
| 通胀3-5% + 经济温和 | 18-25 |
| 通胀2-3% + 经济疲弱 | 12-18 |
| 通胀可控 + 经济正常 | 8-15 |

### 央行购金（制度性需求）

| 行为 | 得分调整 |
|------|----------|
| 中国央行连续增持（月级别，17连增等）| +3到+5 |
| 中国央行单月增持 > 20吨 | +2到+3 |
| 俄罗斯/印度/波兰等国增持 | +1到+2/国 |
| 央行集体净抛售 | -3到-5 |

### 亚洲实物现货溢价 ⚠️ 替代SPDR ETF

> **原理**：ETF代表西方资金（滞后性强，追涨杀跌）。中印实物溢价代表亚洲真实消费需求，是左侧先行信号。

| 情景 | 得分 | 说明 |
|------|------|------|
| 中国现货溢价 > +$30/盎司 | +4到+6 | 🟢 强劲实物需求，机构抄底信号 |
| 中国现货溢价 +$10~$30 | +2到+4 | 实物支撑 |
| 中国现货溢价 < +$10 | 0 | 基本持平 |
| 印度现货贴水 > -$50/盎司 | -1到-3 | 🔴 消费崩塌，金价见顶信号 |
| 印度现货贴水 -$20~$50 | 0 | 基本持平 |
| 中印价差收窄（溢价→贴水）| -3到-5 | 消费结构转变，警惕 |

> **数据来源**：Jin10快讯搜索"现货溢价"或"premium"，或通过 Gold.org、kitco.com 查询

### SPDR ETF 持仓（辅助参考，非主要指标）

| 变化 | 得分调整 |
|------|----------|
| 单日增仓 > 20吨 | +2到+3 |
| 单周增仓 > 50吨 | +3到+5 |
| 持仓持续下降 > 100吨/月 | -2到-4 |
| 持仓基本稳定 | 0 |

> ⚠️ SPDR代表西方资金，具有滞后性。优先参考亚洲现货溢价。

## 六、数据优先级速查

打分时优先获取的数据（按重要性排序）：

1. **美元指数 DXY**（实时）→ 决定美元因子基准分
2. **WTI 原油价格**（盘中实时）→ 滞胀因子核心
3. **10年期TIPS收益率**（实际利率）→ 美元因子 + 滞胀联动
4. **VIX 恐慌指数**（盘中实时）→ 避险因子熔断阈值
5. **地缘最新动态**（Jin10快讯，48小时内对比）→ 地缘动量扣分
6. **中印现货溢价/贴水**（Jin10或kitco）→ 滞胀因子辅助
7. **CPI/PCE 最新值**（月度数据）→ 滞胀因子锚点
8. **Fed官员最新讲话/沃什听证会**（Jin10快讯）→ 政策预期
9. **央行购金数据**（Jin10快讯）→ 制度性需求信号
10. **中国央行17连增等官方数据**（Jin10快讯）→ 制度性需求确认
